-------------------------------------------------------------------------------------

　　　　　　　　　　　　　　　　　　デジタルアシスタント「apu」の使い方

-------------------------------------------------------------------------------------

1.apuとは
apuは、iphoneでいうsiriのようなもので、命令を入力するとその指示通りに動作するデジタルアシスタント機能です。
apuには人工知能が無いため、あらかじめ登録されている86の命令に当てはまるものを入力しないと動作しません。
*Walls Waterloo SP1からはinstr命令を用いています。（例）：「マイコン」という文字が命令文の中にあればマイコンピュータを開く

2.命令一覧
・アプリケーション、アプリケーションを表示、アプリケーションを開いて、アプリ、アプリを表示　→　アプリケーションフォルダを表示する
・マイコンピュータ、マイコンピュータを表示、マイコン、マイコンを開いて、マイコンピュータを開いて　→　マイコンピュータを表示する
・hey apu!、apu　→　はい、なんでしょう？と聞いてくる
・Cドライブを表示、cドライブを表示、Cドライブ、cドライブ、C、c、Cドライブを開いて、cドライブを開いて　→　Cドライブを表示する
・Dドライブを表示、dドライブを表示、Dドライブ、dドライブ、D、d、Dドライブを開いて、dドライブを開いて　→　Dドライブを表示する
・ゲーム、ゲームを表示、ゲームフォルダを表示、ゲームを開いて、ゲームフォルダを開いて　→　ゲームフォルダを表示する
・コントロールパネル、コントロールパネルを表示、コントロールパネルを開いて　→　コントロールパネルを表示する
・壁紙の変更、壁紙を変えたい　→　壁紙の設定を表示する
・インターネット、ネット、ブラウザ、インターネットブラウザ、検索、ブラウザを開いて、インターネットブラウザを開いて　→　Fireboxを開く
・テトリス、テトリスを開いて、テトリスで遊びたい　→　テトリスを開く
・MP、MusicPlayer、MPを開いて、MusicPlayerを開いて、音楽を聞きたい　→　MusicPlayerを開く
・tabexplorer、tabexplorerを実行、タブブラウザ、タブブラウザを実行　→　タブブラウザを開く
・初期化、設定の初期化　→　初期化を行う（要再起動）
・RPG、RPGを開く、RPGで遊びたい　→　史上最悪なRPGを開く
・Wallsの情報、情報　→　「情報」を表示する
・Twirrer、Twitterを開く、ツイッター、つぶやく、つぶやきたい　→　Twitterを開く
・Facebook、Facebookを開く、フェースブック　→　Facebookを開く
・再起動、再起動する　→　再起動
・シャットダウン、シャットダウンする　→　シャットダウン
・ログオフ、ログオフする　→　ログオフ
・スタートメニュー、スタートメニューを開く、スタートメニューの表示　→　スタートメニューの表示
・時報、時報を表示　→　時報を出す
・ah　→　セクシーボイスを流す

ほかにも、「好き」「アホ」「バカ」「馬鹿」「死ね」「役立たず」「かわいい」「ありがとう」「頭悪い」「頭いい」などと入力してみましょう。
